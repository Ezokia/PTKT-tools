Convert to WordPress

Xem chi tiết chức năng converter
  • Ảnh → WordPress image block (auto canh giữa)
  • Video YouTube → WordPress embed block
  • Strip inline styles (font-family, font-size, color, MSO...)
  • Giữ padding/margin-left (thụt lùi đoạn)
  • Clean link patterns (underline-word → clean link)
  • H1 → H2 (font-size 36px), bold paragraphs with font-size → H2
  • Toggle (checkbox hack) → WordPress details block
  • <!--more--> → WordPress more block
  • Footnote/Endnote: giữ nguyên ID links
  • Heading h1 → h2, giữ level cho h3–h6
  • Ảnh → image placeholder trống <figure><img alt=""/></figure>, kèm URL gốc ngay dưới để m làm ảnh
  • Link → apply PTKT style color: #C00000; text-decoration: underline
  • Text wrap <span style="color: #393939"> (inner color override)
  • Drop inline font-size/font-family, giữ color
  • text-align justify/center → WP block attrs + class
  • padding-left → treat as justify
  • Append citation block cuối bài
