WordPress → Blogger Converter

Xem chi tiết chức năng converter
  • Heading (fontSize large) → <h1> với Verdana, x-large, bold
  • Heading (không có fontSize) → <h2>/<h3> với Verdana, bold
  • Paragraph → <p class="MsoNormal"> với Verdana span
  • Image có caption → <table class="tr-caption-container">
  • Image float left (author photo) → <div class="separator"> float left
  • Image thường → <div class="separator"> căn giữa
  • YouTube embed → <iframe> trong <div class="separator">
  • <!--more--> → giữ nguyên
  • Separator → <hr>
  • List → <ul>/<ol> với Verdana spans
  • <strong> → <b>, <em> → <i>
